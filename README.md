# pi-config

Personal [pi](https://github.com/earendil-works/pi-coding-agent) agent configuration.

## Structure

```
~/.pi/agent/
├── keybindings.json           # Keyboard shortcuts
├── decorated-pi.json          # Decorated-pi module toggles
├── mcp-onboarding.json        # MCP server onboarding state
├── extensions/                # Custom extensions (subdirectory per extension)
│   ├── ask-user/
│   ├── copy-all/
│   ├── custom-footer/
│   ├── pi-info/
│   ├── yeet/
│   └── youtube-transcript/
├── themes/
│   └── astrodark.json
├── prompts/                   # Prompt templates (stub)
├── skills/                    # Skills (stub)
├── AGENTS.md                  # Agent instructions for pi
└── README.md
```

Runtime data (sessions, npm packages, auth, settings) stays local at `~/.pi/agent/` and is gitignored.

## Setup

```bash
git clone <remote-url> ~/.pi/agent
```

After editing any config file → run `/reload` in pi.

## Note

- `settings.json` is gitignored — contains provider/model config
- `models.json` is generated by scripts in the [dotfiles](https://github.com/iam/dotfiles) repo, not manually edited
- See `AGENTS.md` for full config details
